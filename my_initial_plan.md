# Create an app that converts Windows cursor format to Linux automatically

- It should be simple to use for non-tech-savvy users.
- It should be easy to install.
- It should have a good UI.
- It should have a good UX.
- It should allow feeding any Windows cursor folder and converting the whole folder to work on Linux automatically.
- It should be self packaged with all of its nessesity so the user shouldnt need to download anyother program to use it.
- at the last it will be an appimage app so it can run on any linux distribution.
